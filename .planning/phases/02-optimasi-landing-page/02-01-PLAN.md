```blade
@if(env('UMAMI_WEBSITE_ID'))

```
UMAMI_WEBSITE_ID=
UMAMI_HOST_URL=http://localhost:3000

  - app/Http/Controllers/DownloadController.php

<action>
1. Di `landing.blade.php`:
   - Tambahkan `data-umami-event="CTA Click"` ke tombol "Dapatkan Sekarang" (CTA di hero section yang submit ke `orders.store`)
   - Tambahkan `data-umami-event="Lead Captured"` ke form submit button lead magnet

2. Di `orders/show.blade.php`:
   - Jika status `verified`, di tombol download tambahkan `data-umami-event="Download Ebook"`

3. Di `lead-magnet/show.blade.php`:
   - Jika ada soft CTA "Lihat Ebook Rp 99.000", tambahkan `data-umami-event="CTA Click"`

4. Di `OrderController@store`:
   - Setelah order berhasil dibuat, jika `UMAMI_WEBSITE_ID` terisi, log event ke Laravel log: `Log::info('Umami Event: Order Created', ['invoice_token' => $order->invoice_token])`

5. Di `LeadMagnetController@store`:
   - Setelah lead berhasil disimpan, log event: `Log::info('Umami Event: Lead Captured', ['email' => $lead->email])`

6. Di `DownloadController@ebook`:
   - Setelah download stream sukses, log event: `Log::info('Umami Event: Download Ebook', ['invoice_token' => $order->invoice_token])`
</action>
<acceptance_criteria>
- Tombol "Dapatkan Sekarang" memiliki `data-umami-event="CTA Click"`
- Tombol submit lead magnet memiliki `data-umami-event="Lead Captured"`
- Tombol download ebook (verified) memiliki `data-umami-event="Download Ebook"`
- Log event tercatat di Laravel log saat aksi terjadi
- Semua test existing tetap passing
</acceptance_criteria>
</task>
