## Summary

Early bird pricing system: 100 pembeli pertama dapat harga spesial Rp 4.900, setelah itu harga naik ke Rp 49.000 (diskon launching), dan Rp 490.000 (harga normal).

### Changes
- **config/products.php:** 3-tier pricing (normal/discount/special) + early bird config
- **OrderController.php:** Server-side price determination berdasarkan `Order::count()`
- **LandingPageController.php:** Dynamic data passing (remaining slots, isEarlyBird, currentPrice)
- **landing.blade.php:** Badge "PROMO 100 PEMBELI PERTAMA — Sisa X slot" + 3-tier comparison UI
- **LeadMagnetController.php:** Dynamic upsell price
- **lead-magnet/show.blade.php:** Dynamic price display (bukan hardcoded)

### Files Modified
- `config/products.php`
- `app/Http/Controllers/OrderController.php`
- `app/Http\Controllers/LandingPageController.php`
- `app/Http/Controllers/LeadMagnetController.php`
- `resources/views/landing.blade.php`
- `resources/views/lead-magnet/show.blade.php`

### How It Works
1. Landing page menampilkan sisa slot early bird dari database
2. Saat user submit order, server hitung `Order::count()` untuk tentukan harga
3. Harga disimpan di database (tidak bisa di-tamper dari client)
4. Setelah 100 order, harga otomatis naik ke Rp 49.000
